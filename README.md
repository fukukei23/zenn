# Zenn Articles

Zenn連携リポジトリ。`master` ブランチへのpushで自動公開。

## 投稿手順

1. `articles/<slug名>.md` を作成（Zennフロントマター必須）
2. `master` へcommit & push
3. Zennが自動検知して公開

## フロントマター

```yaml
---
title: "記事タイトル"
emoji: "😎"
type: "tech" # or "idea"
topics: ["tag1", "tag2"]
published: true  # falseで下書き
---
```
