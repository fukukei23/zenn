設計書 v2 | 作成日: 2026-05-22 | ステータス: ドラフト

GitHub Actions LLM自動生成 人間承認 SNS自動投稿
AIが毎日記事を書いて、あなたは「OK」だけ言えば公開される仕組み
スキャン → ドラフト生成 → 人間確認 → 公開+SNS告知
例えるなら...
AIが「取材記者」→ あなたは「編集長」
記者が毎日ネタを探して原稿を書き、編集長が「掲載OK」を出したら新聞が刷られてSNSでも告知される。そんな仕組みです。
GitHub Actions（ギットハブ アクション）
GitHub上でプログラムを自動実行する仕組み。タイマー設定で毎日決まった時間に動かせる。
Workflow（ワークフロー）
「何を・どんな順番で実行するか」を書いた設定ファイルのこと。
Discord Webhook（ディスコード ウェブフック）
Discordのチャットに自動でメッセージを送る仕組み。
frontmatter（フロントマター）
記事ファイルの先頭にある published: true/false などの設定部分。
slug（スラッグ）
記事のURLになるファイル名部分。ai-agent-self-healing-loop など。

記事生成の仕組み（2段階）

なぜ2段階か
まず安いAIで「ネタ探し」をして、良いネタが見つかったら高いAIで「記事執筆」する。コストを抑えつつ品質を確保する仕組みです。

💡 第1段階: ネタ抽出

AIGLM-5.1（安い）
入力GitHub活動ログ
出力記事候補3件
コスト~$0.002/回

📝 第2段階: 記事執筆

AIClaude Sonnet（高品質）
入力ネタ+ソースコード
出力Zenn記事Markdown
コスト~$0.05/回

ファイル構成

zenn/                           # Zenn記事リポジトリ（GitHub）
├── .github/workflows/
│   ├── discover.yml            # 毎日ネタを探してドラフトを書く
│   └── publish.yml             # 承認されたら公開+SNS投稿
├── scripts/
│   ├── scanner.py              # GitHubリポジトリを巡回するプログラム
│   ├── generator.py            # LLMで記事を書くプログラム
│   ├── notifier.py             # Discordに通知するプログラム
│   ├── publisher.py            # 記事を公開するプログラム
│   ├── sns_poster.py           # SNSに投稿するプログラム
│   ├── requirements.txt        # 必要なPythonライブラリ一覧
│   └── topic_history.json      # 過去に書いたネタの履歴
└── articles/                   # 記事ファイル

必要なAPIキー（GitHub Secretsに登録）

APIキーとは？
サービスを使うための「合言葉」です。GitHubの設定画面に安全に保管し、プログラムからのみ使います。
🤖 ANTHROPIC_API_KEY
記事を書くAI（Claude）の合言葉
📲 BULK_PUBLISH_API_KEY
SNS投稿サービスの合言葉
🔔 DISCORD_WEBHOOK_URL
Discordに通知を送るURL
🔐 PERSONAL_ACCESS_TOKEN
他リポジトリを読むための鍵

既存の未公開記事7本

すでに書いてある7本のドラフトは、このパイプラインとは別に管理します。
公開したい時は published: true に変更してpushするだけで、Workflow BがSNS告知を自動で行います。

設計の選択理由

何を決めたかどうしたかなぜ
承認方法GitHub Issueにコメント記録が残る、スマホからでもできる
通知方法Discordすぐ気付ける、スマホにプッシュ通知
記事を書くAIClaude Sonnet日本語の品質が高い
ネタを探すAIGLM-5.1安い、ネタ探しなら十分
過去何日分見るか7日間1日だと少ない、30日だと多すぎ

制限事項

  • AIが書く記事の品質にはムラがあります（ソースコードの文脈を取りきれない場合あり）
  • 1日1ネタまで（複数欲しい時は手動で連続実行）
  • SNS投稿はテキストのみ（画像はZennが自動生成）
  • BulkPublish無料枠: 100リクエスト/日（1日1投稿なら余裕）
  • GitHub Actions: パブリックリポジトリは無料で無制限
