## はじめに

「24時間動かすAI助手、月額いくらかかる？」

OpenClaw（37万スターのOSS AIアシスタント）をVPSで24時間稼働させて3ヶ月。

結論: モデル使い分け + コスト計算ルールで月額数千円台に抑えられている。

Anthropic Max $220/月との比較も最後に触れる。

## OpenClawのコスト構造

OpenClaw自体は無料（MITライセンス）。

主なコスト: LLM API呼び出し料金。

コストが積み上がる要因:

- **Heartbeat（定期チェック）**: 30分/5分/1分の3層
- **Cronジョブ**: ニュース収集、YouTube要約、バックアップ等
- **サブエージェント**: 49体の専門エージェント
- **チャット応答**: Discord 24時間即応答

VPS代（~$5/月）はAPI料金に比べれば微々たるもの。

## 3モデル使い分け戦略

- **達人 (GLM-5)**: 複雑な分析・推論。コスト高だが品質重視のタスクのみ
- **早人 (GLM-4.7 / Ninja)**: ニュース要約・軽量タスク。**達人より40%安い**
- **匠人 (GPT-5.1-Codex)**: フォールバック用（GLM系が全滅時のみ）

| タスク種別 | 使用モデル | 1回コスト目安 | 月額（1時間ごと実行時） |
|---|---|---|---|
| 軽量（数秒） | GLM-5 | $0.05=8円 | 5,760円 |
| 中量（~30秒） | GLM-5 | $0.20=32円 | 23,040円 |
| 軽量（数秒） | Ninja(GLM-4.7) | $0.03=5円 | 3,600円 |
| 中量（~30秒） | Ninja(GLM-4.7) | $0.12=19円 | 13,680円 |
| 重量（1分~） | Ninja(GLM-4.7) | $0.24=38円 | 27,360円 |

## コスト計算式の導入

新規cron/サブエージェント作成時に**必ず**コスト計算するルールを導入。

計算式:

```
月額実行回数 = 頻度(回/時) × 24時間 × 30日
月額コスト(円) = 月額実行回数 × 1回あたり平均コスト × 為替(160円/$)
```

新規cronの `payload.message` 冒頭に必ず記載:

```
【コスト目安】
- 頻度: 30分ごと → 2回/時
- 月額実行回数: 1,440回
- 概算コスト: $288/月（約46,000円/月）
- 使用モデル: Ninja（軽量タスクのため）
→ ユーザー確認必要
```

## 閾値による自動制御

コストに応じた3段階の承認フロー:

| 月額コスト | 判定 | 必要なアクション |
|---|---|---|
| ~3,000円 | OK | そのまま実装 |
| 3,000〜10,000円 | 要確認 | ユーザーに相談 |
| 10,000円以上 | 承認必須 | ユーザーの明示的承認 |

これをハートビート.mdに書いておくことで、AIエージェント自身がコストを意識してタスク設計する。

## 実際のHeartbeat設計とコスト

3層のHeartbeatとコスト試算:

### 第1層: discord\_context\_scan\_30m（30分ごと）

- Discord会話のTODO・マネタイズ信号をスキャン
- 月額試算: Ninja軽量タスク → 約3,600円
- 判定: ~3,000円 → ちょうど境界なので要確認相当

### 第2層: Config Health Monitor（5分ごと）

- 設定ファイルの変更検知
- 月額試算: 超軽量 → 約2,000円
- 判定: OK

### 第3層: watch\_alert\_tick（1分ごと）

- ファイル変更監視
- 月額試算: 極軽量（ファイル存在チェックのみ）→ 約1,000円
- 判定: OK

### 静穏時間の活用

- 22:00-06:00 JST（8時間）はスキャンをスキップ
- 実行時間を24時間→16時間に圧縮 → コスト33%削減

## Z.AI使用量モニター

`zai-usage-monitor.js` でZ.AIダッシュボードをPuppeteerでスクレイピング。

- 使用量（%）を取得 → Discordに通知
- 閾値: 30%超で注意 / 10%超で警告
- これによりAPI残量を見ながら運用を調整

## Anthropic Maxとの比較

- Anthropic Max: $220/月（Pro 20倍）
- 自前GLM運用: 24時間稼働でも月額数千円台
- z.ai Max: $160/月（Pro 60倍）— Anthropicよりコスパ約3倍
- ただしAnthropic Maxはモデル品質が高い（用途による使い分け）
- 詳細は別記事「AI開発インフラコスト比較」を参照

## コスト削減の5つのコツ

1. **軽量タスクはNinja(GLM-4.7)を使う** — 40%安い
2. **コスト計算式を運用ルールに組み込む** — AI自身が計算
3. **閾値で承認フローを分ける** — 3千/1万円の2段階
4. **静穏時間を設定する** — 22時-6時はスキャンしない
5. **使用量モニターで残量を可視化** — Puppeteer + Discord通知

## まとめ

- 24時間AIアシスタントは工夫次第で月額数千円台で運用可能
- コスト管理を「ルール化」してAI自身に計算させるのがポイント
- GLM-5(達人)とGLM-4.7(早人)の使い分けが最大の節約
- OpenClaw + GLMの組み合わせは、コスト重視の個人運用に最適
